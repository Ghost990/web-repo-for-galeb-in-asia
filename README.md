# Web site repository for Galeb In Asia


## Installation

1. Install [Docker](https://www.docker.com/).

## For development just run
`./dev.sh`

## Deployment

## Build
`docker build -t web-for-galeb-in-asia .`

## Run
`docker run -it --rm -p 80:80 web-for-galeb-in-asia`
